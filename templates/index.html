Understanding The Input

Our enhanced frontend now accepts RAW pricing data (like gold_data.csv) or pre-engineered datasets! The model internally requires 7 exact features: USD Gold, USD/INR, MA10, MA50, EMA10, Volatility, and RSI.

When you upload raw data, our Javascript pipeline automatically strips text/dates, assumes a benchmark exchange rate, and dynamically calculates all Advanced Indicators on the fly before packaging the 60-day Sequence!

Advanced AI Pipeline

Once parsed, the (60, 7) matrix is normalized through the backend Scikit-Learn feature scaler (f_scaler.pkl) before being translated into a sequence vector [1, 60, 7] format and fed into the TensorFlow LSTM sequential architecture.

The final predicted scalar value generated by the AI is then inversely transformed using the attached t_scaler.pkl. This resolves directly to the actual real-world Next Day Gold Price in INR, plotted securely above.

About The ML Model

Architecture: Long Short-Term Memory (LSTM) Neural Network.

Our model is sequence-based machinery designed to ingest the last 60 market days of multi-variate sliding windows to output a single prediction step. Traditional models fail in non-stationary markets, but the internal LSTM cell state effectively tracks long-term dependencies to avoid noise.

Features Trained:

  • Gold_Price_USD (Raw international driver)
  • USD_INR (Exchange benchmark)
  • MA10 & MA50 (Momentum crossovers)
  • EMA10 (Weighted recent activity)
  • Volatility (10-Day Standard Deviation)
  • RSI (Relative Strength Index over 14 periods)

Documentation & Logic

System Workflow:

1. Data Ingestion: The frontend captures either a manual CSV upload or executes an automated live-fetch against Yahoo Finance APIs (`GC=F`, `INR=X`), covering the last 1-year of exact closing histories.

2. Engineering Pipeline: The system automatically cleans missing dates and engineers sophisticated RSI arrays and Variance components instantly in Javascript/Python, mapping raw Ounce metrics smoothly to standard Indian 10-Gram values.

3. Prediction Endpoint: The robust Flask server receives exactly the 60x7 matrix. It scales it against our saved `MinMaxScaler` arrays, triggers the native Keras/TensorFlow prediction weights, and scales the result back out to INR organically.

API Built safely with stateless transaction architecture securely decoupled from the UI layer.
